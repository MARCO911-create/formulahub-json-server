# FormulaHub - JSON Server

Este es el backend para FormulaHub, una plataforma para compartir fórmulas académicas.

## API Endpoints

- GET `/formulas` - Obtener todas las fórmulas
- POST `/formulas` - Crear nueva fórmula
- GET `/formulas/{id}` - Obtener fórmula específica
- PUT `/formulas/{id}` - Actualizar fórmula
- DELETE `/formulas/{id}` - Eliminar fórmula
